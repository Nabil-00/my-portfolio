# My Portfolio

Personal portfolio website built with React and Vite to showcase profile, skills, selected projects, and contact information.

## Sections

- Hero and personal introduction
- About and skills overview
- Project showcase
- Contact section and footer

## Tech Stack

- React
- Vite
- Tailwind CSS
- Vercel Analytics

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Linting

```bash
npm run lint
```
